# YTracker

YTracker is a cross-platform Yandex Tracker client built with Tauri and Rust.

## Features
- View and manage issues from Yandex Tracker
- Create, edit, and comment on issues
- Attach files to issues
- User authentication with OAuth2
- Timer functionality for tracking work time

## Installation
To install YTracker, follow these steps:
Go to the [releases page](https://github.com/Sovego/YTracker/releases) and download the latest version for your operating system.

## Updating
YTracker includes an auto-update feature. When a new version is available, you will be prompted to download and install the update.
